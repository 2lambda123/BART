Detailed Description

template<int dim>
class Iterations< dim >

This class provide an interface between BartDriver and specific iteration schemes.

This class implements an interface between BartDriver<dim> and iteration schemes. The main purpose is such that one would not need to develop different functions according to different problem types.

A generic function to solve the problem for both eigenvalue and fixed-source problems. Inside this function, solving methods will be specified according to Iterations<dim>::is_eigen_problem the boolean.

Note
In fixed-source problems, eig_ptr points to a null pointer and will do nothing.
